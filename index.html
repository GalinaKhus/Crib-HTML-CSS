Шпаргалка по Таблице HTML + CSS

Tеги HTML Описание Атрибуты
<table> Имеет содержимое, которое используется для представления двухмерной таблицы, состоящей из строк и столбцов. Глобальные атрибуты
<thead> Заголовок таблицы, определяет заголовки столбцов таблицы, заключенных в строки таблицы. Глобальные атрибуты
<tbody> Тело таблицы, является семантическим элементом, который будет содержать все данные таблицы, кроме заголовка таблицы и содержимого нижнего колонтитула таблицы. Глобальные атрибуты
<tfoot> Нижний колонтитул таблицы, использует строки таблицы для предоставления содержимого нижнего колонтитула или для суммирования содержимого в конце таблицы. Глобальные атрибуты
<tr> Строка таблицы, используется для добавления строк в таблицу перед добавлением данных таблицы и заголовков таблиц. Глобальные атрибуты
<td> Может быть вложен в элемент строки таблицы, <tr> чтобы добавить ячейку данных в таблицу. colspan и rowspan
<th> Элемент заголовка таблицы, используется для добавления заголовков к строкам и столбцам таблицы и должны быть заключен в элементе строки таблицы, <tr>. colspan и rowspan
Свойства CSS Описание
width Определяетмширину содержимого блока.
border Позволяет одновременно указывать толщину, стиль и цвет границы.
padding Позволяет указать, сколько места должно быть между содержимом элемента и его границами.
text-transform Свойство используется для изменения регистра текста.
text-aling Позволяет контролировать выравнивание текста.
font-size Определяет размер шрифта.
background-color Позволяет указывать цвет фона этого блока.
letter-spcing Позволяет контролировать интервал меж буквами.
empty-cells Eсли в вашей таблице есть пустые ячейки, свойство указывает, следует ли отображать их границы.
border-spacing Устанавливает расстояние между границами соседних ячеек.
border-collapse Свойство устанавливает, имеют ли ячейки внутри общие или отдельные границы.
